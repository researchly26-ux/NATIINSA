Introductory Terms and Concepts

Before we begin our journey through Linux Basics for Hackers, returning to the absolute foundational definitions is critical for navigating the rest of the course.

Binaries

Executable files, similar to Windows .exe files. Found in /usr/bin or /usr/sbin including utilities like ls, cat, and hacking tools like aircrack-ng.

Case Sensitivity

Linux is strictly case-sensitive. Desktop is completely different from desktop. Always check your casing if you get "file not found".

root

The all-powerful superuser. Most hacker tools demand root access since they manipulate network interfaces and system structures directly.

Shell

The environment and interpreter for commands. The default on Kali is usually bash or zsh.

02

A Tour of Kali

Kali Linux is the standard penetration testing distribution, built with hundreds of tools pre-installed.

Log in using the root account username root and the default backward password toor.

The first step in using Kali is clicking the Terminal icon to launch the shell prompt — your primary weapon.

💡 Default Password Warning:

Never leave your Kali password as default. Change it immediately using the passwd command inside the terminal.

03

The Linux Filesystem

Unlike Windows, which uses physical drives (C:, D:) at the base, Linux utilizes a singular logical inverted-tree filesystem starting at the Root directory (/).

/ The absolute root of the filesystem hierarchy.
/root The home directory of the all-powerful root user (different from `/`!).
/etc Contains system-wide configuration files (startup properties, networking, etc).
/home Where standard user directories are stored.
/bin Contains the critical application binaries (the executable files).
04

Basic Commands in Linux

1. pwd (Print Working Directory)

The terminal does not always show your full path. Use this to find your exact location.

kali > pwd

/root

2. whoami

Verify the user identity you are currently operating as.

kali > whoami

root

3. cd (Change Directory)

Navigate the text-based hierarchy using absolute or relative paths.

# Move to the configuration directory

kali > cd /etc

# Move UP one level towards the root

kali:/etc > cd ..

# Jump straight to the top of the tree

kali > cd /

4. ls (List)

List the contents, visibility, and permissions of directories.

# Standard list

kali > ls

bin initrd.img media run var

# Long format list (shows permissions & owners)

kali > ls -l

# Long format AND show hidden files (starts with .)

kali > ls -la

-la

Flags can be combined! -l -a is identical to writing -la.

05

Getting Help

Nearly every command or hacking tool in Linux ships with parameter dictionaries. If you forget your flags, Linux has you covered.

# The help switch (a double-dash for full words)

kali > aircrack-ng --help

# Sometimes tools use single-dash single-letter options

kali > nmap -h

# Open the complete manual pager for a tool

kali > man aircrack-ng

📖 The man Command:

man opens the command in a "pager". Scroll with the arrows or Space, and press q to quit back to the terminal prompt.

06

Finding Stuff

A core skill for a hacker is locating configuration files and exploits hidden across the hierarchy.

locate

Searches a local internal database. Extremely fast but only updates ~once a day.
kali > locate aircrack-ng

whereis

Locates only binary files, their source, and their manual pages.
kali > whereis aircrack-ng

which

Searches STRICTLY inside the directories configured in your $PATH environment variable.
kali > which aircrack-ng

The Almighty find command

The most powerful local search available. It natively traverses directories matching exact conditions.

# Search starting at Root (/) for standard files (-type f) named "apache2"

kali > find / -type f -name apache2

# Target specific directories to speed up queries

kali > find /etc -type f -name apache2

# Use wildcards to find extensions (e.g. .conf)

kali > find /etc -type f -name "apache2.*"

07

Filtering with grep

grep filters bulk text for specific keywords. It's universally used with Pipes (|), which connect the output of one command blindly into the input of another.

# 'ps aux' generates a massive list of running processes

# We pipe it to grep to isolate only the apache server

kali > ps aux | grep apache2

⚠️ Understanding the Pipe:

The pipe key is usually above your ENTER key. It completely redirects the stdout stream of ps aux directly into grep without caching it locally.

08

Modifying Files

Most operations in hacking happen textless without GUI editors. You can use cat (concatenate) with Redirection characters (>) to write files on the fly.

# Creates/Overwrites a file. The prompt halts for input.

kali > cat > hackingskills

Hacking is the most valuable skill set of the 21st century!

# Press CTRL-D to save and exit

# Append to an existing file

kali > cat >> hackingskills

Everyone should learn hacking

# Read the combined file

kali > cat hackingskills

Hacking is the most valuable skill set of the 21st century! Everyone should learn hacking

Redirection Summary:

  • > : Overwrites the target file blindly with the new text.
  • >> : Safely appends text to the absolute bottom of the existing file.
