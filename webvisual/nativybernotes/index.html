IPv4 Class Reference

All IP Classes Explained

🔵
Class A
1.0.0.0 – 126.255.255.255
  • • 8 network bits, 24 host bits
  • • Subnet: 255.0.0.0
  • • Up to 16M hosts per network
  • • Used by large corporations & ISPs
🟢
Class B
128.0.0.0 – 191.255.255.255
  • • 16 network bits, 16 host bits
  • • Subnet: 255.255.0.0
  • • Up to 65K hosts per network
  • • Used by mid-size organizations
🔷
Class C
192.0.0.0 – 223.255.255.255
  • • 24 network bits, 8 host bits
  • • Subnet: 255.255.255.0
  • • Up to 254 hosts per network
  • • Most common — home & small offices
🟣
Class D
224.0.0.0 – 239.255.255.255
  • • Reserved for multicast
  • • No subnet mask
  • • Used in streaming, routing protocols
  • • Not assignable to hosts
🔴
Class E
240.0.0.0 – 255.255.255.255
  • • Reserved / Experimental
  • • Not used in public internet
  • • Research & development only
  • • IANA reserved
🔄
Loopback
127.0.0.0 – 127.255.255.255

Reserved for localhost. Packets never leave the host. Used for testing network software locally.

🏠
Private Ranges
10.x, 172.16–31.x, 192.168.x

Not routable on the public internet. Used in LANs, home routers, VPNs, and corporate networks.

📢
Broadcast
255.255.255.255 / x.x.x.255

Sends packets to all devices on a network. Never assigned to a specific host device.
