What is a Package?

A package is a compressed archive containing software, libraries, configuration files, and metadata. Package managers automate the installation, updating, and removal of software — handling dependencies automatically.

A Package Contains:

  • → Compiled program files (binaries)
  • → Shared libraries
  • → Configuration files
  • → Documentation / man pages
  • → Metadata (version, dependencies)

Package Manager Benefits:

  • ✓ Resolves dependencies automatically
  • ✓ Easy install/remove/update
  • ✓ Verified packages from repos
  • ✓ Keeps system consistent
  • ✓ Security patches via upgrades
02

apt — Debian / Ubuntu / Kali / Parrot

apt stands for Advanced Package Tool. It's the default package manager for Debian-based Linux distributions. It handles downloading, installing, updating, and removing software packages from repositories — online servers that store packages.

Ubuntu Kali Linux Debian Parrot OS
apt commands

sudo apt update # Refresh package list from repositories

sudo apt install <pkg> # Download and install a package

sudo apt remove <pkg> # Remove a package (keep config files)

sudo apt purge <pkg> # Remove package AND config files

sudo apt upgrade # Upgrade all installed packages

sudo apt autoremove # Remove unused dependencies

apt search keyword # Search for packages

apt show <pkg> # Show package details

apt update

Downloads the latest list of available packages from repositories. Does NOT install anything — just refreshes the package index. Always run before installing!

apt install <pkg>

Downloads and installs a package AND all its dependencies automatically. Replace <pkg> with the package name, e.g. nmap, vim, curl.

apt remove <pkg>

Removes a package but keeps its configuration files. Use apt purge to also delete configs.

apt upgrade

Upgrades all installed packages to their latest versions. Always run after apt update to apply security patches.

💡 Golden Rule:

Always run sudo apt update before installing new packages. Without it, you might install outdated versions or miss new packages.

03

pkg — FreeBSD / Termux (Android)

pkg is the package manager for FreeBSD (a Unix-like OS) and Termux (a powerful terminal emulator for Android). It serves the same purpose as apt but for these systems.

FreeBSD Termux (Android)
pkg commands

pkg install <pkg> # Install a package

pkg update && pkg upgrade # Update list AND upgrade all packages

pkg remove <pkg> # Remove a package

pkg search keyword # Search for packages

&&

The AND operator — runs the second command only if the first succeeds. So pkg update && pkg upgrade updates the list first, then upgrades everything only if the update succeeded. Covered in full detail in Module 6!

04

pacman — Arch Linux / Manjaro

pacman is the package manager for Arch Linux and Manjaro. It's known for being simple, efficient, and giving users maximum control. The name is inspired by the Pac-Man video game.

Arch Linux Manjaro
pacman commands

sudo pacman -Syu # Sync + upgrade all packages

sudo pacman -S <pkg> # Install a package

sudo pacman -R <pkg> # Remove a package

sudo pacman -Rns <pkg> # Remove pkg + dependencies + configs

sudo pacman -Ss keyword # Search for packages

sudo pacman -Qi <pkg> # Show detailed package info

sudo pacman -Ql <pkg> # List files installed by package

-S

Stands for "Sync" — syncs with the repository. Used for installing packages.

-Syu

y = refresh database, u = upgrade packages. Together: sync database and upgrade all installed packages.

-R

Stands for "Remove". Removes the specified package.

-Ss

Double s: "Sync Search" — searches for packages matching a keyword in the package name and description.

05

AUR & yay — Arch User Repository

AUR stands for Arch User Repository — a community-driven repository containing user-submitted package build scripts. It allows users to easily build and install software that isn't in the official repositories. yay is an AUR helper that automates this process.

yay (AUR helper)

# Install a package from AUR

yay -S <pkg>

# Search AUR and official repos

yay keyword

# Update everything including AUR packages

yay -Syu

Official Repos vs AUR:

Official Repos

  • • Maintained by Arch team
  • • Security reviewed
  • • Stable and tested

AUR

  • • Community maintained
  • • 50,000+ packages
  • • Review PKGBUILDs before installing
