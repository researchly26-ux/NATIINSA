The Shell

What is a shell? Types of shells — sh, bash, zsh, fish, dash — and all shell-related commands explained in detail.

01

What is a Shell?

A shell is a command-line interpreter that reads your commands and executes them. It's the interface between you and the operating system kernel. The shell processes your input, finds the program you want to run, and executes it.

👤
You
Type commands
🐚
Shell
Interprets & executes
⚙️
Kernel
Controls hardware

💡 Key Point:

The shell is NOT the terminal. The terminal is the window/application you see. The shell is the program running inside that terminal interpreting your commands.

02

Types of Shells

ShellFull NameExplanation
shBourne ShellThe original Unix shell. Very basic and POSIX-compliant. Foundation for all modern shells.
bashBourne Again ShellMost common shell — default on most Linux distros. Extends sh with many powerful features.
zshZ ShellFeature-rich shell with powerful customization. Used with Oh My Zsh + Powerlevel10k themes.
fishFriendly Interactive ShellUser-friendly with great defaults. Modern syntax with syntax highlighting built in. Great for beginners.
dashDebian Almquist ShellLightweight and very fast. Used for system scripts where speed matters more than features.

🏆 Most Common: bash

Default on Ubuntu, Debian, Kali, CentOS, RHEL, and most Linux distros. If you're unsure what shell to learn, start with bash.

⭐ Power User: zsh

Default on macOS. Combined with Oh My Zsh plugin framework offers unmatched customization and productivity.

03

Shell-Related Commands

echo $SHELL # See your current shell

cat /etc/shells # List all installed shells on the system

chsh -s /bin/zsh # Change your default shell to zsh

which bash # Find the path to bash

bash --version # Check bash version

echo $SHELL

Displays the path to your current shell (e.g. /bin/bash). The $ sign indicates a variable. SHELL is an environment variable storing your current shell path.

cat /etc/shells

Shows all available shells installed on the system. The file /etc/shells lists valid login shells.

chsh -s /bin/zsh

chsh = Change Shell. -s = specify the new shell. Changes your default login shell. Takes effect on next login — not immediately!

04

Shell Variables & Environment

The dollar sign ($) indicates a variable in the shell. SHELL is an environment variable — a named container that holds a value accessible throughout your session.

# Key environment variables

echo $SHELL # /bin/bash

echo $HOME # /home/username

echo $PATH # directories searched for commands

echo $USER # current username

env # list ALL environment variables

💡 The PATH variable:

When you type a command like ls, the shell searches each directory in your $PATH for an executable named ls. That's why you can type command names without their full path!
